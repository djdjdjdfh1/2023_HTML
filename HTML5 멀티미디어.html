HTML5 멀티미디어

video

controls : 플레이버튼과 남은시간, 소리 표시
autoplay : 자동실행 (단, muted상태일때)
muted : 무음
loop : 계속 반복

플러그인

object

object 태그는 다양한 파일들을 화면에 출력하는 태그

embed

embed는 object와 동일하게 사용할 수 있다
object 보단 embed가 사용하는게 더 낫다

유튜브 영상들고오기

소스코드로 복사해오면 iframe안에 설정이 되어서 들고올 수 있다
video의 설정을 유튜브에서도 적용할 수 있다
url 뒤에 ?를 이용한 "쿼리스트링"으로 작성할 수 있다
?mute=1&autoplay=1 같은 값으로 들어간다
1은 true 참이라는 값으로 사용, 0은 false 거짓이라는 값으로 사용
